# Mental Health Journal - Frontend

This is the React frontend for the Mental Health Journal application.

## Quick Start

```bash
npm install
npm run dev
```


## Technology Stack

- React 19.1.0
- TypeScript
- Vite 7.0.0
- Microsoft Application Insights

## Development

This project uses Vite for fast development and building. The development server supports hot module replacement (HMR) for instant updates.

### Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build locally
- `npm run lint` - Run ESLint
